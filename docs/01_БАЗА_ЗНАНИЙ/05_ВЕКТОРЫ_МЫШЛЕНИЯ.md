# МАТРИЦА КОГНИТИВНОЙ ТЯГИ (GEARBOX 10V)

Бритва Оккама — это скальпель для отсечения лишнего. Но чтобы перемещаться по информационному пространству и находить решения в запутанных системах, нужен двигатель. Эта матрица описывает **10 фундаментальных векторов** рассудка. Инженерная задача решается переключением между этими векторами (передачами), когда система заходит в тупик.

| ВЕКТОР | ТРИГГЕР (Когда применять) | МЕХАНИКА (Действие) | ИНЖЕНЕРНЫЙ ИМПЛЕМЕНТ (Заземление) |
| :--- | :--- | :--- | :--- |
| **01. Дедукция** | Известен закон, неизвестен результат. | Спуск правила на частный объект. | Применение политики `seccomp` ко всем новым контейнерам. |
| **02. Индукция** | Замечен повторяющийся сбой. | Выведение закона из статистики. | Написание нового линтера на основе 5 последних багов в проде. |
| **03. Абдукция** | Есть "труп", неизвестен убийца. | Поиск гипотезы с макс. вероятностью. | Анализ stacktrace: OOM Killed -> Поиск утечки памяти (дебаггинг). |
| **04. Редукция** | Задача выглядит как "черная магия". | Декомпозиция до базовой физики. | Разбор "Галлюцинаций ИИ" до уровня "Вероятность токена в матрице". |
| **05. Синтез** | Есть инструменты, нет решения. | Эмерджентная сборка деталей. | LLM + Markdown-чеклист = Мета-Конвейер (Новое свойство). |
| **06. Инверсия** | Неизвестен путь к успеху. | Проектирование абсолютного провала. | Составление списка `[MUST_NOT_DO]`. Запрет фатальных антипаттернов. |
| **07. Латераль** | Физический/Ресурсный дедлок. | Уничтожение контекста задачи. | Отказ от `QuickSort` в пользу записи сразу в B-Tree индекс. |
| **08. Парадокс** | Бинарный выбор (A vs B) невозможен. | Удержание взаимоисключений. | Eventual Consistency: БД и врет, и не врет одновременно. |
| **09. Цикличность**| Проблема не имеет финального решения. | Принятие фазового перехода. | Рефакторинг (списание Техдолга) как смена сезонов, а не аврал. |
| **10. Резонанс** | Сила хаоса/атаки превышает ресурсы. | Использование чужой энергии. | Anycast/CDN маршрутизация для рассеивания 10 Tbps DDoS-атаки. |
