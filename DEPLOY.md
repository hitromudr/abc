## 1. Google AI Studio — рекомендуется

**Бесплатно.** Gemini 3.1 Pro — лучшая бесплатная модель для пайплайна. Щедрые лимиты, thinking mode, шаринг по ссылке.

**URL:** https://aistudio.google.com

### Шаги:

1. Залогиньтесь Google-аккаунтом на https://aistudio.google.com
2. Вы в **Playground** — справа выберите модель **Gemini 3.1 Pro Preview**
3. В **System instructions** (справа) вставьте текст промпта из `03_АВТОНОМНЫЙ_ПАЙПЛАЙН.md`
4. Настройки справа:
   - **Temperature:** `0.5`
   - **Thinking level:** `High`
   - **Grounding with Google Search:** убрать (крестик)
5. Слева нажмите **Enable saving** (чтобы промпт сохранился)
6. Протестируйте: напишите `Postgres тормозит под вечер`
7. Ожидаемый результат: Диагностический Протокол Фазы 0 с SQL-запросами + SYSTEM HALT (шаги 2–7 заблокированы)
8. Для шаринга: иконка **Share** вверху справа

### Готовый промпт:

**https://aistudio.google.com/prompts/15dRzT0yfD1XkMtRmeHnfptfBg_ez8mp0**

### Почему рекомендуется:

- Gemini 3.1 Pro — сильный reasoning, корректно выполняет все 7 шагов + Октагон
- Бесплатный tier: до 25 запросов/мин (на порядок щедрее Coze/Poe)
- Thinking mode показывает внутренний процесс модели
- Нет дневных лимитов по кредитам

## 2. Coze (coze.com) — условно бесплатно

**Условно бесплатно.** Публичная ссылка + Telegram/Discord-бот из коробки. Жёсткий дневной лимит кредитов на бесплатном плане (~1–3 запроса/день к GPT-4o).

**URL:** https://www.coze.com

### Шаги:

1. Зарегистрируйтесь на https://www.coze.com
2. В левом меню нажмите **+ Create** → **Create agent** → вкладка **Standard**
3. **Agent name:** `AbraCadabra Architect`
4. **User message billing:** решите, кто платит кредиты (ON = пользователь, OFF = вы)
5. **Agent function description:** `Когнитивный конвейер: 8 фильтров → 6 фаз → Концептуальный Протокол`
6. Нажмите **Confirm**
7. В **Persona & Prompt** вставьте текст промпта из `03_АВТОНОМНЫЙ_ПАЙПЛАЙН.md`
8. Модель: **GPT-4o**
9. **Auto-suggestion:** Off
10. Протестируйте в Preview справа
11. Нажмите **Publish** → выберите платформы:
    - **Bot Store** (публичная ссылка в Coze)
    - **Telegram** (опционально — бот в Telegram)
    - **Discord** (опционально)
12. Скопируйте публичную ссылку

### Бонус: Telegram-бот

Coze позволяет опубликовать бота напрямую в Telegram:
1. При публикации выберите **Telegram**
2. Создайте бота через @BotFather → получите токен
3. Вставьте токен в Coze
4. Бот появится в Telegram — пользователи пишут задачу прямо в чат

### Ограничения:

- Бесплатный план: жёсткий дневной лимит кредитов (может хватить только на 1–3 запроса к GPT-4o)
- Для активного использования нужен Coze Premium

## 3. HuggingChat (huggingface.co/chat) — System Prompt

**Бесплатно.** Open-source модели (Qwen, Llama, DeepSeek), щедрые лимиты. Только для личного использования — публичные ассистенты не поддерживаются.

**URL:** https://huggingface.co/chat

### Шаги:

1. Зарегистрируйтесь / залогиньтесь на https://huggingface.co/chat
2. Слева внизу нажмите **Models** → выберите модель (рекомендуется **Qwen3.5-35B-A3B** или **DeepSeek-R1**)
3. В поле **System Prompt** вставьте текст промпта из `03_АВТОНОМНЫЙ_ПАЙПЛАЙН.md`
4. Нажмите **New chat** и тестируйте

### Ограничения:

- Нет публичных ассистентов — нельзя расшарить бота по ссылке (функция Assistants удалена)
- Модели слабее GPT-4o / Sonnet — могут терять контекст на поздних шагах (5–7)
- Подходит как личный инструмент или fallback

## 4. Custom GPT (ChatGPT) — требует ChatGPT Plus

**Платный** (подписка ChatGPT Plus ~$20/мес для создания, использование бесплатно для всех).

### Шаги:

1. Откройте https://chatgpt.com/gpts/editor (нужен Plus)
2. Вкладка **Configure**
3. **Name:** AbraCadabra Architect
4. **Instructions:** вставьте текст промпта (от `**РОЛЬ:**` до конца шаблона)
5. **Knowledge:** загрузите `03_АВТОНОМНЫЙ_ПАЙПЛАЙН.md`
6. Отключите все Capabilities (Web Browsing, DALL-E, Code Interpreter)
7. **Create** → **Everyone** (публичный)

## Рейтинг моделей для пайплайна

Пайплайн требует сильного reasoning (7 шагов + Октагон). Рейтинг по качеству анализа:

1. **Claude Opus 4.6** — эталон. Платный (Claude Pro / API).
2. **Gemini 3.1 Pro** — ближайший бесплатный конкурент. Google AI Studio, thinking mode, протестирован.
3. **Claude Sonnet 4.6** — бесплатно на claude.ai, хорошее качество.
4. **GPT-4o** — бесплатно в Coze. Стабильный, но слабее на поздних фазах.
5. **DeepSeek-R1** — reasoning-модель, бесплатно. Может удивить на сложных задачах.
6. **Qwen 3.5 35B / Llama 3 70B** — open-source fallback. Теряют контекст на шагах 5–7.
