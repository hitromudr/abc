## [2.0.0] — 2026-03-11

### Added
- Моно-репо abracadabra: **abra** (Архитектор) + **cadabra** (Исполнитель)
- Two-Phase Commit архитектура с Retry Budget и строгой Scope-изоляцией
- `cadabra/core_rules.md` — системный промпт агента-исполнителя (голем + retry budget)
- `EXECUTION_STATE.md` — шаблон машиночитаемого контракта (state machine, DAG, ERROR_LOG, COMPLETION_PROOF)
- `07_ОРКЕСТРАЦИЯ_ЦИКЛА.md` — полный lifecycle abra ↔ cadabra ↔ оператор
- Two-Step Boot протокол для предотвращения коллапса внимания LLM
- Философия GSD, формат ERROR_LOG, примеры контрактов (cadabra/docs)

### Changed
- README v2.0.0: магическая метафора, Quick Start для web-чатов и AI-IDE
- Нейминг модулей: abc/efg/xyz → abra/cadabra

## [1.0.0] — 2026-03-10

### Added
- Первый публичный релиз

- Алгоритм разбора задачи (Фазы 0–6)
- Автономный пайплайн для web-чатов
- Двойная Спираль Разработки (Product & Process)
