## 1. Custom GPT (ChatGPT)

**URL:** https://chatgpt.com → Explore GPTs → Create

### Настройки:

| Поле | Значение |
|---|---|
| **Name** | AbraCadabra Architect |
| **Description** | Когнитивный конвейер для анализа сложных задач. 8 философских фильтров → 6 фаз анализа → Концептуальный Протокол. |
| **Instructions** | Скопируйте содержимое секции `ТЕКСТ СИСТЕМНОГО ПРОМПТА` из `03_АВТОНОМНЫЙ_ПАЙПЛАЙН.md` (строки внутри блока ```text) |
| **Knowledge** | Загрузите файл `03_АВТОНОМНЫЙ_ПАЙПЛАЙН.md` целиком |
| **Capabilities** | Отключите Web Browsing, DALL-E, Code Interpreter (не нужны) |
| **Conversation starters** | `Проанализируй архитектуру системы уведомлений...` / `У нас монолит на Java, нужен план миграции на микросервисы` |

### Шаги:

1. Откройте https://chatgpt.com/gpts/editor
2. Вкладка **Configure**
3. Заполните Name, Description
4. В поле **Instructions** вставьте текст промпта (от `**РОЛЬ:**` до конца шаблона протокола)
5. В **Knowledge** загрузите `03_АВТОНОМНЫЙ_ПАЙПЛАЙН.md`
6. Отключите все Capabilities
7. Нажмите **Create** → выберите **Everyone** (публичный)
8. Скопируйте публичную ссылку

## 2. Coze (coze.com)

**URL:** https://www.coze.com → Create Bot

### Настройки:

| Поле | Значение |
|---|---|
| **Bot name** | AbraCadabra Architect |
| **Bot description** | Когнитивный конвейер: 8 фильтров → 6 фаз → Концептуальный Протокол |
| **Persona & Prompt** | Скопируйте текст промпта из `03_АВТОНОМНЫЙ_ПАЙПЛАЙН.md` |
| **Model** | GPT-4o (или другая доступная) |
| **Plugins** | Не нужны |

### Шаги:

1. Зарегистрируйтесь на https://www.coze.com
2. Нажмите **Create Bot**
3. В **Persona & Prompt** вставьте текст промпта
4. Выберите модель (GPT-4o рекомендуется)
5. Протестируйте в Preview справа
6. Нажмите **Publish** → выберите платформы:
   - **Bot Store** (публичная ссылка в Coze)
   - **Telegram** (опционально — бот в Telegram)
   - **Discord** (опционально)
7. Скопируйте публичную ссылку

### Бонус: Telegram-бот

Coze позволяет опубликовать бота напрямую в Telegram:
1. При публикации выберите **Telegram**
2. Создайте бота через @BotFather → получите токен
3. Вставьте токен в Coze
4. Бот появится в Telegram — пользователи пишут задачу прямо в чат

## 3. HuggingChat Assistants

**URL:** https://huggingface.co/chat → Assistants → Create

### Настройки:

| Поле | Значение |
|---|---|
| **Name** | AbraCadabra Architect |
| **Description** | Когнитивный конвейер для анализа сложных задач |
| **Model** | Qwen/Qwen2.5-72B-Instruct (рекомендуется) или meta-llama/Llama-3.3-70B-Instruct |
| **System prompt** | Скопируйте текст промпта из `03_АВТОНОМНЫЙ_ПАЙПЛАЙН.md` |

### Шаги:

1. Откройте https://huggingface.co/chat/settings/assistants/new
2. Заполните Name, Description
3. Выберите модель (Qwen 2.5 72B держит длинный контекст лучше)
4. В **System prompt** вставьте текст промпта
5. Нажмите **Create**
6. Ассистент появится в списке — скопируйте ссылку

### Ограничения:

- Модели слабее GPT-4o / Sonnet — могут терять контекст на поздних шагах (5–7)
- Нет загрузки файлов в Knowledge — только системный промпт
- Подходит как fallback-вариант

## Рейтинг моделей для пайплайна

Пайплайн требует сильного reasoning (7 шагов + Октагон). Рейтинг по качеству анализа:

1. **Claude Opus 4.6** — эталон. Платный (Claude Pro / API).
2. **Gemini 2.5 Pro** — ближайший бесплатный конкурент. Google AI Studio, 1M контекст.
3. **Claude Sonnet 4.6** — бесплатно на claude.ai, хорошее качество.
4. **GPT-4o** — бесплатно в Custom GPT / Coze. Стабильный, но слабее на поздних фазах.
5. **DeepSeek-R1** — reasoning-модель, бесплатно. Может удивить на сложных задачах.
6. **Qwen 2.5 72B / Llama 3.3 70B** — open-source fallback. Теряют контекст на шагах 5–7.

| **Модель** | GPT-4o | GPT-4o / Gemini | Qwen 72B / Llama 70B |
| **Качество** | Хорошее | Хорошее | Среднее |

| **Лимиты** | ~40 msg/3h (GPT-4o) | Зависит от модели | Без лимитов |
